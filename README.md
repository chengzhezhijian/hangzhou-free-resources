# 全国蹭享地图

全国 **蹭网 · 蹭空调 · 蹭水 · 蹭电** 免费便民资源查询（Top App 级体验）。

**在线访问**：https://chengzhezhijian.github.io/hangzhou-free-resources/?v=32

## 覆盖范围

| 层级 | 内容 | 规模 |
|------|------|------|
| **全国** | 国家政务平台、爱心驿站、公厕/充电地图 | 8+ 官方工具 |
| **72 城** | 图书馆、城市书房、公园、博物馆、政务工具 | 每城 ~40 条 |
| **杭州** | 细粒度收录（书房/纳凉/停车/公园…） | **400+ 条** |
| **合计** | 可筛选点位 + 工具卡片 | **2993 条** |

## 明早验收清单（Premium v32）

### 核心功能
1. 打开首页 → Hero「出门哪儿能蹭网、蹭空调…」+ **四蹭标签**可点
2. 点 📍 选 **北京 / 上海 / 广州** → 列表切换为对应城市
3. 点 **蹭网** → 筛出带 WiFi 的图书馆/书房
4. 点卡片 → 详情 Sheet → **高德地图导航**
5. 底部 Tab：**场景 / 工具 / 说明 / 反馈** 品牌均为「全国蹭享地图」
6. **工具** 页顶部有「全国通用」区块

### Top App 体验（对标 iOS 原生）
7. **向下滚动** → 顶栏与 Hero 紧凑收起（`is-scrolled`）
8. 筛选后 **结果数弹跳**、卡片 **依次入场**
9. 详情 **底部 Sheet 弹簧上滑**、顶部渐变头图
10. 切换城市/分类后 URL 带参数，**可分享**（如 `?city=北京&category=reading`）
