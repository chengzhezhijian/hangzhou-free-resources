# 全国惠民地图

全国 **政府免费便民** 资源查询：自习、纳凉、WiFi、饮水、充电，App 级体验。

**在线访问**：https://chengzhezhijian.github.io/hangzhou-free-resources/?v=33

## 定位文案

| 层级 | 文案 |
|------|------|
| **品牌** | 全国惠民地图 |
| **副标题** | 政府免费便民 · 72城搜完即走 |
| **Hero** | 出门哪儿能免费自习、纳凉？ |
| **引导** | 选城市 · 点设施标签 · 高德导航过去 |
| **设施标签** | 免费WiFi / 有空调 / 可饮水 / 可充电 |

> 从「蹭享」回归「惠民」：用户真实需求是找**政府提供的免费公共服务**，而非网络梗。设施标签用中性、可理解的表述。

## 覆盖范围

| 层级 | 内容 | 规模 |
|------|------|------|
| **全国** | 国家政务平台、爱心驿站、公厕/充电地图 | 8+ 官方工具 |
| **72 城** | 图书馆、城市书房、公园、博物馆、政务工具 | 每城 ~40 条 |
| **杭州** | 细粒度收录（书房/纳凉/停车/公园…） | **400+ 条** |
| **合计** | 可筛选点位 + 工具卡片 | **2993 条** |

## 验收清单

1. 首页 Hero「出门哪儿能免费自习、纳凉？」+ 设施标签可点
2. 点 📍 选 **北京 / 上海 / 广州** → 列表切换
3. 点 **免费WiFi** → 筛出带 WiFi 的图书馆/书房
4. 点卡片 → 详情 → **高德地图导航**
5. 全站品牌「全国惠民地图」，图标「惠」
6. **工具** 页「全国通用」区块在顶部
